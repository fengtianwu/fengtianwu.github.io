1. new env
```
conda create --name <my-env> python=3.7
```


2. list, activate & deactivate env

```
conda env list
conda activate <my-env>
conda deactivate 
```

3. delete env

```
conda env remove --name <my-env>
```

4. install pkg
```
conda install <pkg>
```
